Umikaze
=======

Umikaze is a complete Linux distribution based on Ubuntu and contains
all the necessary software for running or 3D printer or CNC device with
a `Replicape <Replicape>`__. Based on Ubuntu 16.04.2 LTS release for BeagleBone
and optimized for 3D printing.

..  note:: This image distribution was formerly called `Kamikaze`

**Features**

- Redeem : firmware for replicape
- Toggle : firmware for MagnaScreen
- OctoPrint

.. note::

    Octoprint is now installed by installing from source. Enables easier
    hacking and updates to be monitored from within OctoPrints own
    software update system.


**Download** can be found at `Umikaze 2.1.1 <https://github.com/goeland86/Umikaze2/releases/tag/v2.1.1>`_
