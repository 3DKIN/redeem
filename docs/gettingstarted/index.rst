
Getting Started
===============

Required items
~~~~~~~~~~~~~~

-  `Replicape
   Board <https://www.thing-printer.com/product/replicape/>`__
-  `BeagleBone
   Black <https://www.thing-printer.com/product/beaglebone-black-2/>`__
   or a BeagleBone Black Wireless
-  12V to 24V power supply, capable of at least 10A output (more if have
   a heated print bed)
-  3D Printer
-  Monitor with microHDMI to HDMI cable (for troubleshooting)
-  Keyboard (for troubleshooting)

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
